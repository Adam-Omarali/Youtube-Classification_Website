Classifying YouTube Videos with NLP

Predicting if YouTube Videos are Productive or Unproductive

Many of us have experienced falling down the rabbit whole of YouTube, likely because of the intresting but unproductive videos. But the truth is YouTube has a bunch of productive videos with loads of information. The best approach to leverage valuable content is using calssifier.

For my classifier, I used NLP (natural language processing) which is a part of Artificial Intellingence and a similar approach to sentiment-analysis, which can classify statements as positive or negative. Instead, we are classifying if YouTube videos are productive or unproductive based on their title. My model was able to have an accuracy of 85%! This is just one application of NLP, but with so much of the data we have in languages like text and speech, there will be many other applications to come! You may not know it, but Siri uses NLP, which sould tell you we need to do some work on NLP.

How It Works

01. Getting the Data

No body has ever done a project like this before, which also means I had to make my own dataset. One coloumn contained the YouTube title and the other had a 0 or 1 to represent positive or negative. I then read this using pandas.

02. Making the Text Readable

The truth is, computers are not good at understanding words. To help it out, I filter useless words the commonly appeared (of, the, as), converted all the titles to lowercase and finally tokenized the words (makes them indiviual). This allows me to train it on GloVe, a program that can find the relation between words.

03. The Model

I used an LSTM model for the algorithm because it helps store sequential information. It will keep in mind the order of words. Overall, my model had an accuracy of 80%
